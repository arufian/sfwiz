## Critical fixes discovered during M7–M18

- **AliasAccessor.parseAndUpdate doesn't exist** (M7): @salesforce/core 8.x uses `AuthInfo.listAllAuthorizations()` which already includes `aliases[]` on `OrgAuthorization`. Rewrote auth.ts to use that API.
- **AskUserPayload schema** (M7): uses `options: [{label, description}]` not `choices: string[]`. Result has `selected: string | string[]` not `choice`.
- **Salesforce season mapping** (M10): Spring=Jan–Apr, Summer=May–Sep, Winter=Oct–Dec of NEXT year. Fixed from wrong implementation.
- **opentui `fg` not `color`** (M12): `TextBufferOptions` uses `fg?: string | RGBA`. Replaced all `color=` with `fg=` in DirTree.tsx.
- **opentui `border` prop type** (M15): `border={true} borderStyle="single"` not `style={{ border: 'single' }}`.
- **spawnSync timeout missing** (M18): `getSourceTrackingStatus()` had no timeout on spawnSync; test hung >5s. Added `timeout: 4_000`.
- **Bun binary size** (M18): Bun compiled binary includes ~50 MB runtime; actual size 65.6 MB. Adjusted limit to 80 MB.
