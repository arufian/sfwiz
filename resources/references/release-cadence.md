# Salesforce Release Cadence

## Three Releases per Year

| Season | Sandbox Preview | GA Release | Sandbox Update |
|--------|----------------|------------|----------------|
| Spring | January | February | February |
| Summer | May | June | June |
| Winter | October | November | November |

## Release Notes Sources
- https://help.salesforce.com/s/articleView?id=release-notes.salesforce_release_notes.htm
- Sandbox preview orgs receive releases 1 month before GA

## Critical Paths for Developers
1. **Sandbox Preview**: Test new features and validate existing automation
2. **Critical Update**: Review all critical updates; some auto-enable at GA
3. **API Version**: New API version available at each release (v60.0, v61.0, etc.)
4. **Deprecated Features**: Monitor for end-of-life announcements
