# DeleteFilter Enum

DeleteFilter Enum Determines if deleted rows are included in Apex cursor rows.  Enum Values The following are the values of the Database.DeleteFilter enum. Value | Description  
---|---  
DELETED_ROWS_ONLY | Includes only deleted rows.  
NO_DELETED_ROWS | Excludes deleted rows.  
NO_DELETED_SHARING_ROWS | Excludes deleted rows as per sharing rules.  
NO_FILTER | Includes all rows, including deleted rows.
