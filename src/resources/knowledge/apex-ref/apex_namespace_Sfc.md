# Sfc Namespace

Sfc Namespace The Sfc namespace contains classes used in Salesforce Files. The following are the classes in the Sfc namespace.
                                    * **[ContentDownloadContext Enum](atlas.en-us.258.0.apexref.meta/apexref/apex_enum_Sfc_ContentDownloadContext.htm)**  
This enum specifies the download context. 
                                    * **[ContentDownloadHandler Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Sfc_ContentDownloadHandler.htm#apex_class_Sfc_ContentDownloadHandler)**  
Use ContentDownloadHandler to define a custom download handler that controls how content is downloaded.
                                    * **[ContentDownloadHandlerFactory Interface](atlas.en-us.258.0.apexref.meta/apexref/apex_interface_Sfc_ContentDownloadHandlerFactory.htm#apex_interface_Sfc_ContentDownloadHandlerFactory)**  
Use this interface to provide a class factory that Salesforce can call to create instances of your custom ContentDownloadHandler.
