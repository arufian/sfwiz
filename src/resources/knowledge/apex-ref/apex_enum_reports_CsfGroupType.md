# CsfGroupType Enum

CsfGroupType Enum The group level at which the custom summary format aggregate is displayed in a report. Enum Values The following are the values of the Reports.CsfGroupType enum. Value | Description  
---|---  
ALL | The aggregate is displayed at the end of every summary row.  
CUSTOM | The aggregate is displayed at specified grouping levels.  
GRAND_TOTAL | The aggregate is displayed only at the grand total level.
