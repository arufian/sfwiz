# RenderContext Interface

RenderContext Interface A wrapper interface that is used to retrieve application and environment context information. Namespace [Canvas](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Canvas.htm "The Canvas namespace provides an interface and classes for canvas apps in Salesforce.") Usage Use this interface to retrieve application and environment context information for your canvas app. For this interface, you don’t need to create an implementation. Use the default implementation that Salesforce provides.
