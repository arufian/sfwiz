# AnalyticsCloudComponentLayoutItem Class

AnalyticsCloudComponentLayoutItem Class Represents the settings for a Wave Analytics dashboard on a standard or custom page. Namespace [Metadata](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Metadata.htm "The Metadata namespace provides classes and methods for working with custom metadata in Salesforce") Usage Use this class when accessing [Metadata.Layout](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_class_Metadata_Layout "Represents the metadata associated with a page layout.") metadata components. For more information, see “AnalyticsCloudComponentLayoutItem” in the [Metadata API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.258.0.api_meta.meta/api_meta/meta_intro.htm).

AnalyticsCloudComponentLayoutItem Properties The following are properties for AnalyticsCloudComponentLayoutItem.
            * **[assetType](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_assetType)**  
Specifies the type of Wave Analytics asset.
            * **[devName](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_devName)**  
Unique development name of the dashboard to add.
            * **[error](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_error)**  
An error string that is populated only when an error occurred in the underlying dashboard.
            * **[filter](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_filter)**  
Dashboard filters for mapping data fields in the dashboard to the object’s fields.
            * **[height](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_height)**  
Specifies the height of the dashboard, in pixels.
            * **[hideOnError](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_hideOnError)**  
Controls whether users see a dashboard that has an error.
            * **[showHeader](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_showHeader)**  
If true, includes the dashboard’s header bar. If false, the dashboard appears without a header bar.
            * **[showSharing](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_showSharing)**  
If set to true, and the dashboard is shareable the dashboard shows the Share icon. If set to false, the dashboard doesn’t show the Share icon.
            * **[showTitle](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_showTitle)**  
If true, includes the dashboard’s title above the dashboard. If false, the dashboard appears without a title.
            * **[width](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_AnalyticsCloudComponentLayoutItem.htm#apex_Metadata_AnalyticsCloudComponentLayoutItem_width)**  
Specifies the width of the dashboard, in pixels or percentage.
