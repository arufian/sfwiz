# SidebarComponent Class

SidebarComponent Class Represents a specific custom console component to display in a container that hosts multiple components in one of the sidebars of the Salesforce console. Namespace [Metadata](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Metadata.htm "The Metadata namespace provides classes and methods for working with custom metadata in Salesforce") Usage Use this class when accessing [Metadata.Layout](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_class_Metadata_Layout "Represents the metadata associated with a page layout.") metadata components. For more information, see “SidebarComponent” in the [Metadata API Developer Guide](https://developer.salesforce.com/docs/atlas.en-us.258.0.api_meta.meta/api_meta/meta_intro.htm).

SidebarComponent Properties The following are properties for SidebarComponent.
            * **[componentType](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_componentType)**  
Specifies the component type. Valid values are “KnowledgeOne”, “Lookup”, “Milestones”, “RelatedList”, “Topics”, “Files”, and “CaseExperts”. 
            * **[createAction](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_createAction)**  
If the component is a lookup field, the name of the quick action used to create a record.
            * **[enableLinking](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_enableLinking)**  
If the component is a lookup field, lets users associate a record with this field.
            * **[height](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_height)**  
The height of the component in the container. The unit property determines the unit of measurement, in pixels or percent.
            * **[knowledgeOneEnable](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_knowledgeOneEnable)**  
Indicates if the component is enabled for Knowledge One.
            * **[label](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_label)**  
The name of the component as it displays to console users. Available for components in a container with the style of tabs or accordion.
            * **[lookup](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_lookup)**  
If the component is a lookup field, the name of the field.
            * **[page_x](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_page_x)**  
If the component is a Visualforce page, the name of the Visualforce page.
            * **[relatedLists](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_relatedLists)**  
If the component is a related list component, the list of related list names.
            * **[unit](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_unit)**  
The unit of measurement (pixels or percent) for the height and width of the component in the container.
            * **[updateAction](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_updateAction)**  
If the component is a lookup field, the name of the quick action used to update a record.
            * **[width](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_SidebarComponent.htm#apex_Metadata_SidebarComponent_width)**  
The width of the component in the container. The unit property determines the unit of measurement, in pixels or percent.
