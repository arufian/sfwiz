# Process Namespace

Process Namespace The Process namespace provides an interface and classes for passing data between your organization and a flow. The following are the interfaces and classes in the Process namespace.
                                    * **[Plugin Interface](atlas.en-us.258.0.apexref.meta/apexref/apex_interface_Process_Plugin.htm#apex_interface_Process_Plugin)**  
Allows you to pass data between your organization and a specified flow.
                                    * **[PluginDescribeResult Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Process_PluginDescribeResult.htm#apex_class_Process_PluginDescribeResult)**  
Describes the input and output parameters for Process.PluginResult.
                                    * **[PluginDescribeResult.InputParameter Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Process_PluginDescribeResult_InputParameter.htm#apex_class_Process_PluginDescribeResult_InputParameter)**  
Describes the input parameter for Process.PluginResult.
                                    * **[PluginDescribeResult.OutputParameter Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Process_PluginDescribeResult_OutputParameter.htm#apex_class_Process_PluginDescribeResult_OutputParameter)**  
Describes the output parameter for Process.PluginResult.
                                    * **[PluginDescribeResult.ParameterType Enum](atlas.en-us.258.0.apexref.meta/apexref/apex_enum_Process_PluginDescribeResult_ParameterType.htm)**  
Specifies the data types of input and output parameters of the Process.PluginDescribeResult class.
                                    * **[PluginRequest Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Process_PluginRequest.htm#apex_class_Process_PluginRequest)**  
Passes input parameters from the class that implements the Process.Plugin interface to the flow.
                                    * **[PluginResult Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Process_PluginResult.htm#apex_class_Process_PluginResult)**  
Returns output parameters from the class that implements the Process.Plugin interface to the flow.
