# FeedLayoutFilterPosition Enum

FeedLayoutFilterPosition Enum Describes where the feed filters list is included in the layout. Enum Values The following are the values of the Metadata.FeedLayoutFilterPosition enum. Value | Description  
---|---  
CenterDropDown | As a drop-down list in the center column.  
LeftFixed | As a fixed list in the left column.  
LeftFloat | As a floating list in the left column.
