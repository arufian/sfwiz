# DeployMessage Class

DeployMessage Class Represents result information for the deployment of a metadata component. Namespace [Metadata](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Metadata.htm "The Metadata namespace provides classes and methods for working with custom metadata in Salesforce") Usage Use DeployMessage to access detailed information about component deployments. Salesforce provides a list of DeployMessages for a completed deployment via the DeployDetails and DeployResults instances sent in the [DeployCallback.handleResult()](atlas.en-us.258.0.apexref.meta/apexref/apex_interface_Metadata_DeployCallback.htm#apex_Metadata_DeployCallback_handleResult "Method that is called when the asynchronous deployment of custom metadata completes.") callback.

DeployMessage Properties The following are properties for DeployMessage.
            * **[changed](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_changed)**  
Determines whether the component was changed after deployment. If true, the component was changed as a result of the deployment. If false, the deployed component was the same as the corresponding component already in the org.
            * **[columnNumber](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_columnNumber)**  
Each component is represented by a text file. If an error occurs during deployment, this property represents the column of the text file where the error occurred.
            * **[componentType](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_componentType)**  
The metadata type of the component in the deployment.
            * **[created](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_created)**  
If true, the component was created as a result of the deployment. If false, the component was modified as a result of the deployment.
            * **[createdDate](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_createdDate)**  
The date and time when the component was created as a result of the deployment.
            * **[deleted](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_deleted)**  
If true, the component was deleted as a result of the deployment. If false, the component was either new or modified as result of the deployment.
            * **[fileName](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_fileName)**  
The name of the file in the metadata archive used to deploy the component.
            * **[fullName](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_fullName)**  
Full name for the custom metadata component.
            * **[id](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_id)**  
ID of the component that was deployed.
            * **[lineNumber](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_lineNumber)**  
Each component is represented by a text file. If an error occurs during deployment, this field represents the line number of the text file where the error occurred.
            * **[problem](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_problem)**  
If an error or warning occurred, this field contains a description of the problem that caused the deployment to fail.
            * **[problemType](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_problemType)**  
Indicates the problem type, for example, an error or warning.
            * **[success](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_DeployMessage.htm#apex_Metadata_DeployMessage_success)**  
Indicates whether the component was successfully deployed (true) or not (false).
