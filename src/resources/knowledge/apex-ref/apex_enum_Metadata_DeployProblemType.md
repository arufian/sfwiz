# DeployProblemType Enum

DeployProblemType Enum Describes the problem type for an unsuccessful component deploy. Enum Values The following are the values of the Metadata.DeployProblemType enum. Value | Description  
---|---  
Error | The deploy problem is an error.  
Info | The deploy problem is of type “Info”.  
Warning | The deploy problem is a warning.
