# getDeveloperName()

getDeveloperName() Retrieves the internal API name of the canvas app. Signature public String getDeveloperName() Return Value Type: [String](atlas.en-us.258.0.apexref.meta/apexref/apex_methods_system_string.htm#apex_methods_system_string "Contains methods for the String primitive data type.") Usage Use this method to get the API name of the canvas app. You specify this value in the API Name field when you expose the canvas app by creating a connected app.
