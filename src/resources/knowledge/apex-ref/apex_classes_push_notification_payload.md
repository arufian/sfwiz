# PushNotificationPayload Class

PushNotificationPayload Class Contains methods to create the notification message payload for an Apple device. Namespace [Messaging](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Messaging.htm "The Messaging namespace provides classes and methods for Salesforce outbound and inbound email functionality.") Usage Apple has specific requirements for the notification payload. and this class has helper methods to create the payload. For more information on Apple’s payload parameters, search for “Apple Push Notification Service” at <https://developer.apple.com/library/mac/documentation/>. Example See the [Push Notification Example](atlas.en-us.258.0.apexref.meta/apexref/apex_classes_push_notification.htm#push_notification_example_code).
