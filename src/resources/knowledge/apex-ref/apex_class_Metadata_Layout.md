# Layout Class

Layout Class Represents the metadata associated with a page layout. Namespace [Metadata](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Metadata.htm "The Metadata namespace provides classes and methods for working with custom metadata in Salesforce") Usage Use this class to access layout metadata components. For more information, see [Layout](https://developer.salesforce.com/docs/atlas.en-us.258.0.api_meta.meta/api_meta/meta_layouts.htm) in the Metadata API Developer Guide.

            * **[Layout Methods](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_methods)**  

Layout Properties The following are properties for Layout.
            * **[customButtons](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_customButtons)**  
The custom buttons for this layout.
            * **[customConsoleComponents](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_customConsoleComponents)**  
Represents custom console components (Visualforce pages, lookup fields, or related lists) on a page layout.
            * **[emailDefault](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_emailDefault)**  
Default value for the email checkbox. Only relevant if the showEmailCheckbox property is set.
            * **[excludeButtons](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_excludeButtons)**  
List of standard buttons to exclude from this layout.
            * **[feedLayout](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_feedLayout)**  
Represents the values that define the feed view of a feed-based page layout.
            * **[headers](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_headers)**  
Represents the layout headers used for tagging.
            * **[layoutSections](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_layoutSections)**  
The main sections of the layout containing fields, s-controls, and custom links. The order here determines the layout order.
            * **[miniLayout](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_miniLayout)**  
Represents a minilayout, which is used in the mini view of a record in the Console tab, hover details, and event overlays.
            * **[multilineLayoutFields](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_multilineLayoutFields)**  
Fields for special multiline layout fields which appear in OpportunityProduct layouts.
            * **[platformActionList](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_platformActionList)**  
The list of actions, and their order, that display in the Salesforce mobile action bar for the layout.
            * **[quickActionList](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_quickActionList)**  
The list of quick actions that display in the full Salesforce site for the page layout.
            * **[relatedContent](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_relatedContent)**  
The Related Content section of the page layout. 
            * **[relatedLists](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_relatedLists)**  
The related lists for the layout, listed in the order they appear in the user interface.
            * **[relatedObjects](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_relatedObjects)**  
The list of related objects that appears in the mini view of the console.
            * **[runAssignmentRulesDefault](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_runAssignmentRulesDefault)**  
Default value for the “run assignment rules” checkbox. Only relevant if the showRunAssignmentRulesCheckbox property is set.
            * **[showEmailCheckbox](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_showEmailCheckbox)**  
Controls whether to show the email checkbox. Only allowed on Case, CaseClose, and Task layouts. The default state of checkbox is controlled by the emailDefault property.
            * **[showHighlightsPanel](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_showHighlightsPanel)**  
If set, the highlights panel displays on pages in the Salesforce console.
            * **[showInteractionLogPanel](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_showInteractionLogPanel)**  
If set, the interaction log displays on pages in the Salesforce console.
            * **[showKnowledgeComponent](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_showKnowledgeComponent)**  
Only allowed on Case layouts. If set, the Knowledge sidebar displays on cases in the Salesforce console.
            * **[showRunAssignmentRulesCheckbox](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_showRunAssignmentRulesCheckbox)**  
Controls whether to show the Run Assignment Rules checkbox. Only allowed on Lead and Case layouts. The default state of checkbox is controlled by the runAssignmentRulesDefault property.
            * **[showSolutionSection](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_showSolutionSection)**  
Only allowed on CaseClose layout. If set, the built-in solution information section shows up on the page.
            * **[showSubmitAndAttachButton](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_showSubmitAndAttachButton)**  
For Cast layouts only. If set, the Submit & Add Attachment button displays on case edit pages to portal users in the Customer Portal.
            * **[summaryLayout](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Metadata_Layout.htm#apex_Metadata_Layout_summaryLayout)**  
The summary layout for this layout.
