# MatchRecord Class

            * **[getAdditionalInformation()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Datacloud_MatchRecord.htm#apex_Datacloud_MatchRecord_getAdditionalInformation)**

            * **[getMatchConfidence()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Datacloud_MatchRecord.htm#apex_Datacloud_MatchRecord_getMatchConfidence)**
