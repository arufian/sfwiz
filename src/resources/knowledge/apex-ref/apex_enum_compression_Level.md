# Level Enum

Level Enum Specifies the compression level for creating a zip file. Usage Use Level enum with the getLevel() and setLevel(value) methods in the ZipWriter class. Enum Values The following are the values of the Compression.Level enum. Value | Description  
---|---  
BEST_COMPRESSION | Compression level for best compression.  
BEST_SPEED | Compression level for fastest compression.  
DEFAULT_LEVEL | Default compression level.  
NO_COMPRESSION | Compression level for no compression.
