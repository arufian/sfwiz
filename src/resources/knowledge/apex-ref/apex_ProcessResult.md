# ProcessResult Class

ProcessResult Class After you submit a record for approval, use the ProcessResult class to process the results of an approval process. Namespace [Approval](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Approval.htm "The Approval namespace provides classes and methods for approval processes.") Usage A ProcessResult object is returned by the process method. You must specify the Approval namespace when creating an instance of this class. For example:
[code] Approval.ProcessResult result = Approval.process(req1);
[/code]

ProcessResult Methods The following are methods for ProcessResult. All are instance methods.
          * **[getEntityId()](atlas.en-us.258.0.apexref.meta/apexref/apex_ProcessResult.htm#apex_Approval_ProcessResult_getEntityId)**  
The ID of the record being processed.
          * **[getErrors()](atlas.en-us.258.0.apexref.meta/apexref/apex_ProcessResult.htm#apex_Approval_ProcessResult_getErrors)**  
If an error occurred, returns an array of one or more database error objects including the error code and description.
          * **[getInstanceId()](atlas.en-us.258.0.apexref.meta/apexref/apex_ProcessResult.htm#apex_Approval_ProcessResult_getInstanceId)**  
The ID of the approval process that has been submitted for approval.
          * **[getInstanceStatus()](atlas.en-us.258.0.apexref.meta/apexref/apex_ProcessResult.htm#apex_Approval_ProcessResult_getInstanceStatus)**  
The status of the current approval process. Valid values are: Approved, Rejected, Removed or Pending.
          * **[getNewWorkitemIds()](atlas.en-us.258.0.apexref.meta/apexref/apex_ProcessResult.htm#apex_Approval_ProcessResult_getNewWorkitemIds)**  
The IDs of the new items submitted to the approval process. There can be 0 or 1 approval processes. 
          * **[isSuccess()](atlas.en-us.258.0.apexref.meta/apexref/apex_ProcessResult.htm#apex_Approval_ProcessResult_isSuccess)**  
A Boolean value that is set to true if the approval process completed successfully; otherwise, it is set to false.
