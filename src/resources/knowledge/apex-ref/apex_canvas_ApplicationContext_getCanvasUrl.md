# getCanvasUrl()

getCanvasUrl() Retrieves the fully qualified URL of the canvas app. Signature public String getCanvasUrl() Return Value Type: [String](atlas.en-us.258.0.apexref.meta/apexref/apex_methods_system_string.htm#apex_methods_system_string "Contains methods for the String primitive data type.") Usage Use this method to get the URL of the canvas app, for example: http://instance.salesforce.com:8080/canvas_app_path/canvas_app.jsp.
