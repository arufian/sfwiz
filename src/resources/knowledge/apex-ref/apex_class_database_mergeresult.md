# MergeResult Class

MergeResult Class Contains the result of a merge Database method operation. Namespace [Database](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Database.htm "The Database namespace provides classes used with DML operations.") Usage The Database.merge method returns a Database.MergeResult object for each merged record. MergeResult Methods The following are methods for MergeResult. All are instance methods.
              * **[getErrors()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_database_mergeresult.htm#apex_Database_MergeResult_getErrors)**  
Returns a list of Database.Error objects representing the errors encountered, if any, during a merge operation using the Database.merge method. If no error occurred, returns null.
              * **[getId()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_database_mergeresult.htm#apex_Database_MergeResult_getId)**  
Returns the ID of the master record into which other records were merged.
              * **[getMergedRecordIds()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_database_mergeresult.htm#apex_Database_MergeResult_getMergedRecordIds)**  
Returns the IDs of the records merged into the master record.
              * **[getUpdatedRelatedIds()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_database_mergeresult.htm#apex_Database_MergeResult_getUpdatedRelatedIds)**  
Returns the IDs of all related records that were reparented as a result of the merge that are viewable by the user sending the merge call.
              * **[isSuccess()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_database_mergeresult.htm#apex_Database_MergeResult_isSuccess)**  
Indicates whether the merge was successful (true) or not (false).
