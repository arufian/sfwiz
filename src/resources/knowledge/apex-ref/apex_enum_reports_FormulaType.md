# FormulaType Enum

FormulaType Enum The format of the numbers in a custom summary formula. Enum Values The following are the values of the Reports.FormulaType enum. Value | Description  
---|---  
CURRENCY | Formatted as currency. For example, $100.00.  
NUMBER | Formatted as numbers. For example, 100.  
PERCENT | Formatted as percentages. For example, 100%.
