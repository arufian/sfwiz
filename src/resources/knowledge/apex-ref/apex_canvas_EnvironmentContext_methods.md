# EnvironmentContext Methods

EnvironmentContext Methods The following are methods for EnvironmentContext.
          * **[addEntityField(fieldName)](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_addEntityField.htm)**  
Adds a field to the list of object fields that are returned in the signed request Record object when the component appears on a Visualforce page that’s placed on an object.
          * **[addEntityFields(fieldNames)](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_addEntityFields.htm)**  
Adds a set of fields to the list of object fields that are returned in the signed request Record object when the component appears on a Visualforce page that’s placed on an object.
          * **[getDisplayLocation()](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_getDisplayLocation.htm)**  
Retrieves the display location where the canvas app is being called from. For example, a value of Visualforce page.
          * **[getEntityFields()](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_getEntityFields.htm)**  
Retrieves the list of object fields that are returned in the signed request Record object when the component appears on a Visualforce page that’s placed on an object.
          * **[getLocationUrl()](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_getLocationUrl.htm)**  
Retrieves the location URL of the canvas app.
          * **[getParametersAsJSON()](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_getParametersAsJSON.htm)**  
Retrieves the current custom parameters for the canvas app. Parameters are returned as a JSON string.
          * **[getSublocation()](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_getSublocation.htm)**  
Retrieves the display sublocation where the canvas app is being called from.
          * **[setParametersAsJSON(jsonString)](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_EnvironmentContext_setParametersAsJSON.htm)**  
Sets the custom parameters for the canvas app.
