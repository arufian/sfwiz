# Support Namespace

Support Namespace The Support namespace provides an interface used for Case Feed. The following is the interface in the Support namespace.
                                    * **[EmailTemplateSelector Interface](atlas.en-us.258.0.apexref.meta/apexref/apex_email_template_selector.htm#apex_email_template_selector)**  
The Support.EmailTemplateSelector interface enables providing default email templates in Case Feed. With default email templates, specified email templates are preloaded for cases based on criteria such as case origin or subject. 
                                    * **[MilestoneTriggerTimeCalculator Interface](atlas.en-us.258.0.apexref.meta/apexref/apex_interface_support_milestonetriggertimecalculator.htm#apex_interface_support_milestonetriggertimecalculator)**  
The Support.MilestoneTriggerTimeCalculator interface calculates the time trigger for a milestone.
