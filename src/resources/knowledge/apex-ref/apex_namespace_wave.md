# Wave Namespace

Wave Namespace The classes in the Wave namespace are part of the CRM Analytics Analytics SDK, designed to facilitate querying CRM Analytics data from Apex code. The following are the classes in the Wave namespace.
                                    * **[QueryBuilder Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_wave_QueryBuilder.htm#apex_class_wave_QueryBuilder)**  
The QueryBuilder class provides methods for constructing well-formed SAQL queries to pass to CRM Analytics.
                                    * **[QueryNode Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_wave_QueryNode.htm#apex_class_wave_QueryNode)**  
Define each node of the query - such as projection, groups, order, filters. Execute the query.
                                    * **[ProjectionNode Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_wave_ProjectionNode.htm#apex_class_wave_ProjectionNode)**  
Add aggregate functions to the query, or define an alias.
                                    * **[Templates Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Wave_Templates.htm#apex_class_Wave_Templates)**  
The Templates class provides methods for retrieving CRM Analytics template collections, individual templates, and template configurations.
                                    * **[TemplatesSearchOptions Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Wave_TemplatesSearchOptions.htm#apex_class_Wave_TemplatesSearchOptions)**  
The TemplatesSearchOptions class provides optional properties to filter the template collection.
