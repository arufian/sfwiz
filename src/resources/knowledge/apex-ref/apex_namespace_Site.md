# Site Namespace

Site Namespace The Site namespace provides an interface for rewriting Sites URLs. The following is the interface in the Site namespace.
                                    * **[UrlRewriter Interface](atlas.en-us.258.0.apexref.meta/apexref/apex_site_urlRewriter_reference.htm#apex_site_urlRewriter_reference)**  
Enables rewriting Sites URLs.
                                    * **[Site Exceptions](atlas.en-us.258.0.apexref.meta/apexref/apex_Site_exceptions.htm)**  
The Site namespace contains an exception class.
