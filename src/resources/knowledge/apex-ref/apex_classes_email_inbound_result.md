# InboundEmailResult Class

InboundEmailResult Class The InboundEmailResult object is used to return the result of the email service. If this object is null, the result is assumed to be successful. Namespace [Messaging](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Messaging.htm "The Messaging namespace provides classes and methods for Salesforce outbound and inbound email functionality.")
