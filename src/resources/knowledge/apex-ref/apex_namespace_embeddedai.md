# embeddedai Namespace

embeddedai Namespace The embeddedai namespace provides classes and methods to manage and represent records and data in Apex to support embedded AI features. These are the classes in the embeddedai namespace.
                                    * **[ApexMap Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_embeddedai_ApexMap.htm#apex_class_embeddedai_ApexMap)**  
Create, clone, and convert string based key-value pairs to a JSON string format.
                                    * **[RecordApexRepresentation Class](atlas.en-us.258.0.apexref.meta/apexref/apex_class_embeddedai_RecordApexRepresentation.htm#apex_class_embeddedai_RecordApexRepresentation)**  
Contains properties and a method to create a serializable representation of a record and its associated data for AI service integration and data processing.
