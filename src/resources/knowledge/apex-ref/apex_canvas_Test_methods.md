# Test Methods

Test Methods The Test class provides methods for creating test contexts and invoking your CanvasLifecycleHandler with mock data. The following are methods for Test. All are static methods.
          * **[mockRenderContext(applicationContextTestValues, environmentContextTestValues)](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_Test_mockRenderContext.htm)**  
Creates and returns a test Canvas.RenderContext based on the provided application and environment context parameters.
          * **[testCanvasLifecycle(lifecycleHandler, mockRenderContext)](atlas.en-us.258.0.apexref.meta/apexref/apex_canvas_Test_testCanvasLifecycle.htm)**  
Calls the Canvas test framework to invoke a CanvasLifecycleHandler with the provided RenderContext.
