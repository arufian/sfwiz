# PostAuthorizationRequest Constructors

PostAuthorizationRequest Constructors Lists the constructors for postauthorization requests.  The following are constructors for PostAuthorizationRequest.
                    * **[PostAuthorizationRequest(amount)](atlas.en-us.258.0.apexref.meta/apexref/apex_commercepayments_PostAuthorizationRequest_ctor.htm)**  
Constructor for building the amount in a postauthorization request. This constructor is intended for test usage and throws an exception if used outside of the Apex test context.
