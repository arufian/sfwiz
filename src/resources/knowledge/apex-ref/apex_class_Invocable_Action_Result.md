# Action.Result Class

Action.Result Class Contains methods to retrieve results from invocable actions called from Apex code. Namespace [Invocable](atlas.en-us.258.0.apexref.meta/apexref/apex_namespace_Invocable.htm "The Invocable namespace provides classes for calling invocable actions from Apex.")

            * **[clone()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Invocable_Action_Result.htm#apex_Invocable_Action_Result_clone)**

Gets a list of errors that were returned by an invocable action.
            * **[getInvocationParameters()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Invocable_Action_Result.htm#apex_Invocable_Action_Result_getInvocationParameters)**  
Gets a list of the parameter values set for an invocable action. This method returns a list that contains the input parameter values for each invocation of an action. Each map in the list contains a key for the name of each input parameter.
            * **[getOutputParameters()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Invocable_Action_Result.htm#apex_Invocable_Action_Result_getOutputParameters)**  
Gets a list of the parameter values returned by an invocable action. This method returns a list that contains the result for each invocation of an action. Each map in the list contains a key for the name of each output parameter.
            * **[isSuccess()](atlas.en-us.258.0.apexref.meta/apexref/apex_class_Invocable_Action_Result.htm#apex_Invocable_Action_Result_isSuccess)**  
Determines if an invocable action ran without errors.
